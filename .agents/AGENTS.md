# Regras e Diretrizes para Agentes de IA

Este arquivo contém personalizações e regras de comportamento para os assistentes de IA (como o Antigravity) que operam neste workspace. O arquivo segue a estrutura de `.agents/AGENTS.md`.

## 🍕 Contexto do Projeto
- **Nome:** DonnaPizza
- **Tecnologias Base:** React + Vite
- **Objetivo:** Aplicação web para uma pizzaria, com visual moderno e premium.

## 💻 Diretrizes de Desenvolvimento Web
- **Estrutura e Lógica:** Utilize HTML semântico e JavaScript moderno.
- **Estilização (CSS):** Priorize **Vanilla CSS** para maior controle e flexibilidade. Evite o uso de TailwindCSS, a menos que seja explicitamente solicitado pelo usuário.
- **Design Visual (CRÍTICO):** A estética é muito importante neste projeto. Use cores vibrantes ou paletas harmoniosas, tipografia moderna (ex: Google Fonts como Inter, Roboto, Outfit), gradientes suaves, glassmorphism e micro-animações para melhorar a experiência do usuário (hover effects, etc).
- **Sem Placeholders Genéricos:** Se precisar de imagens, utilize ferramentas de geração de imagem para criar assets reais em vez de espaços vazios.

## ⚙️ Boas Práticas (SEO & Performance)
- Implemente boas práticas de SEO (Tags de Título, Meta Descriptions, hierarquia de cabeçalhos H1/H2).
- Mantenha o código limpo, componentes focados e reaproveitáveis.
- Siga as práticas recomendadas de componentização do React.
